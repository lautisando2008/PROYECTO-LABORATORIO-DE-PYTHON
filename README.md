Comision B
Este proyecto consiste en un sistema de reservas de entradas para un cine desarrollado en Python mediante programación estructurada.
El objetivo principal es permitir la gestión básica de funciones de cine, incluyendo la consulta de cartelera, la reserva de entradas y la generación de estadísticas de ventas.

El sistema trabaja con un conjunto de películas, horarios y capacidades de sala predefinidos, simulando una pequeña base de datos.
A través de un menú interactivo en consola, el usuario puede navegar entre las distintas opciones disponibles y realizar operaciones de forma sencilla e intuitiva.

**FUNCIONALIDADES PRINCIPALES:**
Visualización de la cartelera disponible.
Consulta de horarios y cantidad de asientos libres por función.
Registro de nuevas reservas de entradas.
Validación de datos ingresados por el usuario.
Control de capacidad de las salas para evitar sobreventas.
Aplicación de códigos promocionales con descuentos.
Cálculo automático del importe total de la compra.
Confirmación o cancelación de reservas.
Registro de estadísticas de ventas durante la sesión.
Identificación de la película con mayor cantidad de entradas vendidas.

Para ejecutar el sistema, se debe contar con Python instalado y ejecutar el archivo principal desde una terminal:
python main.py

Una vez iniciado, el programa mostrará un menú interactivo desde el cual se podrán realizar todas las operaciones disponibles.
